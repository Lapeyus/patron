## 1. Resumen Ejecutivo

Se propone una solucion web privada, responsive (movil y desktop), para gestion y exhibicion de perfiles con contenido multimedia (fotos y videos), filtros avanzados, control de acceso y contacto directo por WhatsApp.

La propuesta cubre:
- Implementacion inicial completa.
- Puesta en produccion.
- Mantenimiento mensual y gestion de perfiles.
- Hosting y dominio a costo real.

---

## 2. Alcance del Servicio

### Incluye
- Catalogo web con perfiles, fotos y videos.
- Vista detallada por perfil con informacion estructurada.
- Busqueda y filtros por criterios clave (edad, ubicacion, categorias y metadatos).
- Flujo de contacto directo por WhatsApp con mensaje preconfigurado.
- Control de acceso por password para restringir visualizacion.
- Interfaz bilingue (espanol/ingles).
- Soporte para rese√±as y etiquetas de perfil.
- Soporte tecnico y mantenimiento continuo.

### No incluye en esta fase
- Backend con autenticacion robusta de servidor.
- Panel administrativo dedicado tipo CMS.
- Integraciones con sistemas externos (CRM, ERP, pasarelas, etc.).
